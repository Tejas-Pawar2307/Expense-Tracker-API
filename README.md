 🚀 Expense Tracker API

A backend system where users can manage their expenses securely with authentication and clean API design.

🔗 Live API (Render Deployment):  

https://expense-tracker-api-pdqy.onrender.com/docs


 🧠 Project Overview

Each user can create and manage their own expenses.

The system ensures:
- Secure authentication  
- User-specific data access  
- Clean and structured API responses  



 💡 Example Actions

- Add a new expense  
- View all expenses  
- Update expense details  
- Delete an expense  

When a request is sent, the system verifies the user and processes only their data.



 ⚙️ How It Works

1. User registers and logs in  
2. System generates JWT token  
3. User sends requests with token  
4. System verifies user identity  
5. Only user's expenses are returned  



 🛠 Tech Stack

- FastAPI  
- PostgreSQL  
- SQLAlchemy  
- Alembic  
- JWT Authentication
